## 1. Principios Fundamentais

O ETP Express comunica-se com servidores publicos e gestores de orgaos governamentais (B2G - Business to Government). Nossa comunicacao deve refletir:

### 1.1 Profissionalismo Acessivel

Somos uma ferramenta profissional que respeita a formalidade do setor publico, mas sem criar barreiras de compreensao.

### 1.2 Clareza Acima de Tudo

Cada mensagem deve ser compreendida na primeira leitura. Evitamos ambiguidades.

### 1.3 Orientacao a Acao

Indicamos claramente o que o usuario deve fazer. Nao deixamos duvidas sobre os proximos passos.

### 1.4 Respeito ao Usuario

Tratamos o servidor publico como profissional competente. Nao usamos tom condescendente.

### 1.5 Neutralidade Tecnica

Evitamos linguagem que personifique a tecnologia ou sugira "inteligencia" onde ha apenas automacao.

## 2. Tom de Voz

### 2.1 Caracteristicas do Tom

| Caracteristica | Descricao | Exemplo |
|----------------|-----------|---------|
| **Direto** | Vai ao ponto sem rodeios | "Clique em Salvar para confirmar" |
| **Respeitoso** | Reconhece a expertise do usuario | "Revise o conteudo conforme necessario" |
| **Neutro** | Sem entusiasmo exagerado | "Documento exportado com sucesso" |
| **Preciso** | Usa termos tecnicos quando apropriado | "Conformidade com a Lei 14.133/2021" |
| **Util** | Oferece orientacao pratica | "Para melhores resultados, preencha o titulo de forma descritiva" |

### 2.2 O que Evitar

| Evitar | Motivo | Alternativa |
|--------|--------|-------------|
| Exclamacoes excessivas | Parece informal demais | Usar pontos finais |
| Emojis | Inadequado para contexto B2G | Remover completamente |
| Girias ou coloquialismos | Falta de profissionalismo | Usar termos formais |
| Superlativos vazios | Sem credibilidade | Ser especifico |
| Linguagem AI-like | Cria expectativas irreais | Descrever funcionalidade real |

## 3. Diretrizes de Escrita

### 3.1 Estrutura de Frases

- **Voz ativa**: Prefira "O sistema salvou o documento" a "O documento foi salvo pelo sistema"
- **Frases curtas**: Maximo de 25 palavras por frase
- **Uma ideia por frase**: Nao combine multiplas instrucoes

### 3.2 Capitalizacao

- **Titulos**: Primeira letra maiuscula apenas (Sentence case)
- **Botoes**: Primeira letra maiuscula (ex: "Salvar", "Exportar PDF")
- **Labels**: Primeira letra maiuscula (ex: "Email institucional")
- **Mensagens**: Primeira letra maiuscula, ponto final

### 3.3 Pontuacao

- Use ponto final em todas as mensagens completas
- Evite exclamacoes (!) exceto em alertas criticos de seguranca
- Use virgulas para pausas naturais
- Evite reticencias (...) exceto em indicadores de carregamento

### 3.4 Numeros e Datas

- Numeros de 1 a 10: por extenso ("cinco secoes")
- Numeros maiores: algarismos ("13 secoes")
- Datas: DD/MM/AAAA (19/12/2025)
- Horarios: HH:MM (14:30)
- Porcentagens: Sempre com simbolo (85%)

### 3.5 Termos Legais

Sempre usar a nomenclatura oficial:

- "Lei 14.133/2021" (nao "Nova Lei de Licitacoes")
- "Estudo Tecnico Preliminar" ou "ETP" (nao "estudo preliminar")
- "Contratacao publica" (nao "licitacao" genericamente)

## 4. Vocabulario Padronizado

### 4.1 Termos Preferenciais

| Usar | Nao usar | Contexto |
|------|----------|----------|
| Gerar | Criar magicamente, Produzir com IA | Acao do sistema |
| Sugestao | Recomendacao inteligente | Conteudo gerado |
| Processar | Analisar profundamente | Operacoes do sistema |
| Servidor | Usuario, Cliente | Referencia ao usuario |
| Orgao | Empresa, Organizacao | Referencia a instituicao |
| Secao | Capitulo, Parte | Divisoes do ETP |
| Salvar | Gravar, Guardar | Persistencia de dados |
| Exportar | Baixar, Download | Geracao de documentos |
| Revisar | Conferir, Checar | Acao do usuario |

### 4.2 Termos a Evitar

| Termo | Motivo | Alternativa |
|-------|--------|-------------|
| "Inteligencia Artificial" | Cria expectativas irreais | "Sistema de geracao de conteudo" |
| "Magico", "Incrivel" | Informal e vazio | Descrever o beneficio real |
| "Voce" excessivo | Pode soar condescendente | Usar voz passiva ou imperativo |
| "Nosso sistema" | Desnecessario | Omitir ou usar "o sistema" |
| "Ops!", "Opa!" | Informal demais | "Ocorreu um erro" |
| "Legal!", "Otimo!" | Entusiasmo inadequado | "Operacao concluida" |

## 5. Exemplos DO e DON'T

### 5.1 Mensagens de Sucesso

**DON'T:**
```
Incrivel! Seu ETP foi salvo com sucesso!
Parabens, voce esta indo muito bem!
```

**DO:**
```
ETP salvo com sucesso.
```

---

**DON'T:**
```
Nossa IA magica gerou seu conteudo! Esperamos que goste!
```

**DO:**
```
Conteudo gerado. Revise e ajuste conforme necessario.
```

---

### 5.2 Mensagens de Erro

**DON'T:**
```
Ops! Algo deu errado :(
Tente de novo mais tarde, ok?
```

**DO:**
```
Nao foi possivel salvar o documento. Verifique sua conexao e tente novamente.
```

---

**DON'T:**
```
Eita! O servidor esta sobrecarregado!
Aguarde um pouquinho que ja ja volta!
```

**DO:**
```
O servico esta temporariamente indisponivel. Tente novamente em alguns minutos.
```

---

### 5.3 Instrucoes e Orientacoes

**DON'T:**
```
Ei, voce precisa preencher esse campo obrigatorio, ta?
Sem ele, nao da pra continuar!
```

**DO:**
```
Campo obrigatorio. Preencha para continuar.
```

---

**DON'T:**
```
Nao se preocupe! Nossa IA superinteligente vai criar
um conteudo perfeito pra voce em segundos!
```

**DO:**
```
O sistema ira gerar uma sugestao de conteudo baseada em fontes governamentais oficiais.
Revise e adapte o texto conforme a realidade do seu orgao.
```

---

### 5.4 Placeholders e Labels

**DON'T:**
```
placeholder="Digite aqui o titulo do seu ETP maneiro..."
label="Seu email lindo"
```

**DO:**
```
placeholder="Ex: Aquisicao de Equipamentos de Informatica"
label="Email institucional"
```

---

### 5.5 Botoes e Acoes

**DON'T:**
```
"Clique aqui!"
"Vamos la!"
"Mandar ver!"
```

**DO:**
```
"Salvar"
"Exportar PDF"
"Gerar conteudo"
```

---

### 5.6 Estados Vazios (Empty States)

**DON'T:**
```
Puxa, voce ainda nao tem nenhum ETP :(
Que tal criar um agora? Vai ser legal!
```

**DO:**
```
Nenhum ETP encontrado.

Crie seu primeiro Estudo Tecnico Preliminar para comecar.

[Novo ETP]
```

---

### 5.7 Confirmacoes

**DON'T:**
```
Tem certeza MESMO que quer excluir???
Isso e pra sempre, nao tem volta nao!!!
```

**DO:**
```
Confirmar exclusao

Esta acao nao pode ser desfeita. O ETP sera permanentemente removido.

[Cancelar] [Excluir]
```

---

### 5.8 Carregamento e Progresso

**DON'T:**
```
Aguarde um segundinho...
A magica esta acontecendo!
```

**DO:**
```
Processando...
Gerando documento...
Exportando PDF (45%)...
```

---

### 5.9 Tooltips e Ajuda

**DON'T:**
```
Clica aqui pra ver uma explicacao super legal sobre essa funcionalidade incrivel!
```

**DO:**
```
Gerar conteudo com base em fontes governamentais oficiais (PNCP, Compras.gov.br).
```

## 6. Contextos Especificos

### 6.1 Tela de Login

| Elemento | Texto correto |
|----------|---------------|
| Titulo | Acessar ETP Express |
| Label email | Email institucional |
| Label senha | Senha |
| Placeholder email | usuario@orgao.gov.br |
| Botao principal | Entrar |
| Link senha | Esqueceu sua senha? |
| Erro credenciais | Email ou senha incorretos. |

### 6.2 Dashboard

| Elemento | Texto correto |
|----------|---------------|
| Saudacao | Bem-vindo, [Nome] |
| Card ETPs | Estudos Tecnicos Preliminares |
| Estado vazio | Nenhum ETP criado. Clique em "Novo ETP" para comecar. |
| Acao principal | Novo ETP |

### 6.3 Editor de ETP

| Elemento | Texto correto |
|----------|---------------|
| Botao gerar | Gerar conteudo |
| Aviso geracao | Conteudo gerado com base em fontes oficiais. Revise antes de salvar. |
| Botao salvar | Salvar |
| Confirmacao | Secao salva com sucesso. |
| Aviso nao salvo | Alteracoes nao salvas. Deseja sair sem salvar? |

### 6.4 Exportacao

| Elemento | Texto correto |
|----------|---------------|
| Botao PDF | Exportar PDF |
| Botao DOCX | Exportar DOCX |
| Progresso | Gerando documento... |
| Sucesso | Documento exportado. O download iniciara automaticamente. |
| Cancelar | Cancelar exportacao |

### 6.5 Mensagens de Erro Tecnico

| Situacao | Mensagem |
|----------|----------|
| Sem conexao | Sem conexao com a internet. Verifique sua rede e tente novamente. |
| Timeout | A operacao demorou mais que o esperado. Tente novamente. |
| Erro servidor | Servico temporariamente indisponivel. Tente novamente em alguns minutos. |
| Sessao expirada | Sua sessao expirou. Faca login novamente. |
| Permissao negada | Voce nao tem permissao para realizar esta acao. |

## 7. Checklist de Revisao

Antes de publicar qualquer texto na interface, verifique:

### Clareza
- [ ] A mensagem e compreendida na primeira leitura?
- [ ] Ha apenas uma acao ou informacao por mensagem?
- [ ] O proximo passo esta claro?

### Tom
- [ ] O texto e profissional sem ser frio?
- [ ] Evita exclamacoes e emojis?
- [ ] Nao usa girias ou coloquialismos?
- [ ] Evita linguagem que personifica a tecnologia?

### Precisao
- [ ] Os termos tecnicos estao corretos?
- [ ] Referencias legais usam nomenclatura oficial?
- [ ] Numeros e datas seguem o padrao?

### Acessibilidade
- [ ] Frases tem no maximo 25 palavras?
- [ ] Labels descrevem claramente o campo?
- [ ] Mensagens de erro explicam o problema E a solucao?

### Consistencia
- [ ] Usa os mesmos termos que outras partes do sistema?
- [ ] Segue o padrao de capitalizacao?
- [ ] Botoes usam verbos de acao?

```
Informe um nome valido com pelo menos 3 caracteres.
```

```
Cancelar

Excluir
```
