## 2. Introdução

O ETP Express ("nós", "nosso" ou "Sistema") está comprometido com a proteção da privacidade e dos dados pessoais de seus usuários. Esta Política de Privacidade descreve como coletamos, usamos, armazenamos, compartilhamos e protegemos suas informações pessoais em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei 13.709/2018).

Ao utilizar o ETP Express, você concorda com as práticas descritas nesta política. Recomendamos a leitura atenta deste documento.

## 6. Compartilhamento de Dados

### 6.1 Provedores de Inteligência Artificial

Para fornecer as funcionalidades de geração de conteúdo, compartilhamos dados com os seguintes provedores:

| Provedor | País | Dados Compartilhados | Finalidade |
|----------|------|---------------------|------------|
| **OpenAI** | Estados Unidos | Contexto dos ETPs, prompts | Geração de texto com IA |
| **Perplexity** | Estados Unidos | Queries de pesquisa, contexto | Pesquisa de contratos similares |

**Importante:**
- Dados pessoais diretos (nome, email) **NÃO** são enviados a esses provedores
- Apenas o conteúdo institucional necessário para geração é compartilhado
- Esses provedores possuem políticas de não-retenção para chamadas de API

### 6.2 Infraestrutura

| Provedor | País | Serviço |
|----------|------|---------|
| **Railway** | Estados Unidos | Hospedagem da aplicação e banco de dados |

### 6.3 Autoridades

Podemos compartilhar seus dados com autoridades competentes quando:
- Obrigados por lei ou ordem judicial
- Necessário para proteção de direitos do ETP Express
- Investigação de fraudes ou violações de segurança

## 7. Transferência Internacional de Dados

Seus dados são transferidos para os Estados Unidos da América, onde estão localizados os servidores de nossos provedores de serviço:

- **Railway** (hospedagem e banco de dados)
- **OpenAI** (geração de texto)
- **Perplexity** (pesquisa)

Esta transferência é realizada com base no seu **consentimento específico** (LGPD Art. 33, VIII), obtido durante o cadastro no sistema.

**Garantias aplicadas:**
- Os provedores mantêm padrões de segurança adequados
- Dados são transmitidos via conexões criptografadas (HTTPS/TLS)
- Contratos incluem cláusulas de proteção de dados

---

## 8. Retenção de Dados

Seus dados são retidos pelos seguintes períodos:

| Tipo de Dado | Período de Retenção | Justificativa Legal |
|--------------|--------------------|--------------------|
| Dados de conta | Enquanto conta ativa + 5 anos | Código Civil Art. 206 |
| ETPs e seções | Enquanto conta ativa | Execução do contrato |
| Logs de auditoria | 90 dias | Boas práticas de segurança |
| Analytics | 1 ano (anonimizado após 30 dias) | Legítimo interesse |
| Backups | 30 dias | Recuperação de desastres |

Após o término da retenção, os dados são eliminados ou anonimizados de forma irreversível.

## 10. Segurança dos Dados

Implementamos medidas técnicas e organizacionais para proteger seus dados:

### 10.1 Medidas Técnicas
- **Criptografia em trânsito:** HTTPS/TLS para todas as comunicações
- **Criptografia em repouso:** PostgreSQL encryption
- **Hashing de senhas:** bcrypt com salt
- **Tokens JWT:** Autenticação segura com expiração
- **Rate limiting:** Proteção contra ataques de força bruta

### 10.2 Medidas Organizacionais
- Acesso restrito a dados por necessidade
- Logs de auditoria de acessos
- Procedimentos de resposta a incidentes
- Rotação periódica de chaves e secrets

### 10.3 Detecção de Anomalias
- Monitoramento de padrões de acesso anômalos
- Alertas automáticos para comportamentos suspeitos
- Audit trail completo de acessos a dados sensíveis

## 12. Menores de Idade

O ETP Express é destinado a servidores públicos e profissionais. Não coletamos intencionalmente dados de menores de 18 anos. Se tomarmos conhecimento de dados de menores, eles serão eliminados imediatamente.
